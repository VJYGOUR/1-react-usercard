## Overview
This project demonstrates a **reusable React component** (`UserCard`) that receives data through **props** and allows users to toggle their **online/offline status**. It showcases **state management, props usage, and component reusability** in React.

---

## Features
- Display user information: **name** and **email**
- Show **online/offline status** using a color indicator and emoji
- **Toggle status** with a button or by clicking the card
- Reusable `UserCard` component used for multiple users
- Clean, styled card layout

---

## Tech Stack
- React 18+  
- TypeScript (for type safety)  
- CSS-in-JS (inline styles for simplicity)
